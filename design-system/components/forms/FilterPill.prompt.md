The signature Zillow filter-bar pill — white with a chevron, turns blue when a filter is applied, optional count badge. Clicking opens a popover (you render the popover separately).

```jsx
<FilterPill label="For Sale" />
<FilterPill label="Home Type" applied count={3} />
<FilterPill label="Price" open />
```

Use a horizontal row of these under the search input on results pages, ending with a primary "Save search" Button.
