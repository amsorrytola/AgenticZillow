Small chip for feature tags, counts, AI value badges and inline status.

```jsx
<Badge tone="blue" variant="soft">~6% under Zestimate</Badge>
<Badge tone="success" variant="outline">Great deal</Badge>
<Badge tone="neutral" variant="soft">EV charger</Badge>
```

`tone`: neutral / blue / success / warning / error / violet. `variant`: solid / soft / outline.
